# LangGraph Chatbot

This is a simple AI chatbot built using **LangGraph** and **Cohere API**.

## Features
- Uses LangGraph to manage chatbot flow
- Integrates Cohere LLM for generating responses
- CLI-based chat interface
- Maintains conversation state
